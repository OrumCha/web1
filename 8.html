Structure of Internet

인터넷은 본질적으로 2대의 컴퓨터끼리 상호작용하는 것이고, 2대의 컴퓨터의 역할을 각각 구분한다.
web server와 web browser라는 프로그램을 각각 만들어서
각 컴퓨터에서 동작하도록 만들고, 제작한 웹페이지를 web server에 저장한다.
  1. web browser(client computer, web client)
    2. -web server에 웹페이지의 파일을 요청한다.

  2. web server(server computer, web server)
    3. -web brower가 요청한 파일을 찾아서 응답한다.
