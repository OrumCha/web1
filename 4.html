Structure of Web

  1. title
    2. -웹페이지의 제목; 검색 엔진이 웹페이지 분석할때 가장 중요하게 생각하는 태그
  2. meta
    3. -영어가 아닌 문자로 웹페이지를 만들면 문자가 깨질 수도 있어서 utf-8의 방식으로 작성했다면, utf-8로 읽어야한다는 정보를 줄 수 있다.
  3. head
    4. -본문을 설명하는 태그; title과 meta 포함
  4. body
    5. -본문
  5. html
    6. -body태그와 head태그를 감싸는 하나의 태그
  6. !doctype html
    7. -html로서 만들어졌다는 것을 표현하기 위해 문서 시작에 추가되는 코드
